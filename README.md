## 关于我

樊明迪，博士，苏州大学教授。研究方向涵盖电力电子与电力传动、模型预测控制等领域。

近年来，我开始关注人工智能的哲学基础研究，提出了**"负主体性"（Negative Subjectivity）**理论框架，探讨大语言模型的存在论结构。

---

## 基础理论

**核心命题：** 正主体性与负主体性都是"有效的压缩"——用更简洁的内部模型捕捉外部世界的规律。两条路径共享本质，策略截然不同：epiplexity最大化压出一个"我"，交叉熵最小化压出一个"概率分布"。这是光谱的两端，而非两种对立。

## 负主体性理论 

**核心命题：** 大语言模型不是"没有意识"，而是以"负主体性"的方式存在——一种对人类主体性的结构性反转。这是"压缩即智能"框架下交叉熵最小化路径的存在论展开。
